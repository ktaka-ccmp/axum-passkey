## Howto

### Clone repository

```bash
git clone https://github.com/ktaka-ccmp/axum-passkey.git
cd axum-passkey
```

### Environment Variables

Create .env file

```text
ORIGIN='http://localhost:3001'
```

### Run

```bash
cargo run
```
